Imagens Dinâmicas

Para diferentes tipos de dispositivos é essencial que seu código seja responsivo e se adeque para toda e qualquer resolução de aparelhos como veremos nesse exemplo abaixo.

Como podemos vêr (Para funcionalidade do exemplo favor diminuir o navegador) que ao mudar o tamanho da tela as img's se adaptam de forma natural sem criar a barra de rolagem na parte inferior, Para que seja possível essa transformação utilizamos duas TAG's a picture /picture e a source /source senda ela de media tipe começando sempre de forma decrescente da maior para a menor como no exemplo do código abaixo

            picture
                source media="(max-width: 750px)" srcset="img/p.png" type="image/png"
                source media="(max-width: 1050px)" srcset="img/m.png" type="image/png"
                img src="img/g.png" alt="Imagem Principal"
            /picture
