Código fonte

O comando document class="getElementby"(TESTE) é escrito em JavaScript.
para melhor viasualização usamos o codigo code /code de (de forma semantica) para espaçar as fontes de forma mono espaçada correta. Exemplo sem identação

num = int(imput('digite um número')) if num % 2 == 0 print(f' O número {num} é PAR') else: print(f' o número {num} e IMPAR') print('Fim do programa')


Fazendo a uitilizaçlão da TAG pre /pre o codigo exemplificado entra em identação de forma natural Ex:

                
num = int(imput('digite um número'))
if num % 2 == 0
    print(f' O número {num} é PAR')
else:
    print(f' o número {num} e IMPAR')
print('Fim do programa')

OBS: Fazendo a utilização do atalho SHIFT+TAB toda a linha de "Codigo" escrita no exemplo toca alinhado ao lado lateral esquerdo um espaçamento por vez.
